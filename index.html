03. Projects

Prosthetic Hand

Robotic Prosthetic Hand

Designed and built a functional robotic prosthetic hand using a Raspberry Pi Pico. Integrated flex sensors into a glove to measure finger movement via resistance changes, mapping each sensor to servo motors to replicate natural hand motion. Future expansion planned using myoelectric (EMG) sensors.

  • Flex Sensors
